# Database-InformationSystems-Project
University project in Database Systems &amp; Information Engineering, including ERD, schema design, SQL queries and system analysis.
## How to Run the Project

1. Install dependencies:
   pip install flask

2. Import the database:
   Run the SQL script `mytau012.sql` in your MySQL/MariaDB environment.

3. Start the server:
   python main.py

4. Open your browser at:
   http://127.0.0.1:5000
